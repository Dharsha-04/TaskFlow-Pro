# TaskFlow-Pro
To-Do List

A modern and responsive productivity dashboard built using HTML, CSS, JavaScript, Bootstrap 5, and Chart.js. TaskFlow Pro helps users manage tasks, track goals, monitor productivity, and visualize progress through an elegant dark-themed interface.

📌 Features
📋 Task Management
Add new tasks
Edit existing tasks
Delete tasks
Mark tasks as completed
Set task priority levels:
High
Medium
Low
Add task descriptions
Assign due dates
Add session timings (Start Time & End Time)

🔍 Search & Filter
Search tasks instantly
Filter tasks by:
All Tasks
Completed
Pending
High Priority
Medium Priority
Low Priority

📊 Dashboard
Total Tasks Counter
Completed Tasks Counter
Pending Tasks Counter
Goal Progress Percentage
Dynamic Progress Bar
Recent Tasks Section

🎯 Goal Tracking
Create productivity goals
Set target percentages
Delete goals
Track progress visually

📈 Analytics
Task Completion Doughnut Chart
Priority Distribution Bar Chart
Real-time dashboard updates
Analytics summary section

💾 Local Storage
Tasks persist after browser refresh
Goals persist after browser refresh
No database required

🌙 Modern Dark UI
Fully responsive design
Mobile-friendly layout
Smooth animations
Custom dark theme
Modern dashboard cards

Enter:
Task Title
Description
Date
Start Time
End Time
Priority

Goal Name

Low Priority Tasks
